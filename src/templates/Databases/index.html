Databases

Uploading a database

Go to 'Upload' to upload a database. You can upload zip files, files with an .csv extension or SQL dumps. Adding a name for the table is a must and we can add an optional description.

In case of a zip file with multiple tables you will be redirected to a join table page where you need to join the tables in your zip file.

My databases

After succesfully uploading a database we can now see our database under this tab.

If we want to go to our table we press on the name. We can also edit and delete our database.
