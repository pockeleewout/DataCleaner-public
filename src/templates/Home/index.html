Database cleaning tool made by students of the University of Antwerp for their class "Programming project databases"

Tutorial

Uploading a database

Go to the page Databases -> Upload a database. Here you can upload a .csv file or zip file or SQL dump. In case of a zip file with multiple tables you will be redirected a join table page where you are able to join the tables.

Viewing your table

Go to the page Tables If you press on the name of your database you should be redirected to the page of your table.

Settings

Go to the page Settings Here you should be able too find the personal information of your account if you are logged in and here you are able to edit your information.
